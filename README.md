# FLORIDA SHOP

플로리다 쇼핑몰 전용 Next.js 프로젝트입니다.

## 로컬 실행

```bash
pnpm install
pnpm dev
```

- 기본 홈 진입(`/`)은 자동으로 `/florida`로 이동합니다.
- 브라우저에서 `http://localhost:3000/florida`로 접속해 확인할 수 있습니다.

## 주요 화면

- `/florida` : 쇼핑몰 메인 홈
- `/florida/all` : 전체 상품/카테고리
- `/florida/product/[id]` : 상품 상세
- `/florida/cart` : 장바구니
- `/florida/checkout` : 결제
- `/florida/mypage` : 주문/찜/최근 본 상품
